自定义组件功能

1. 字段顺序 fields: []
2. 字段名称 field.name
3. 是否必填 field.required
4. 是否启用 field.enabled
5. 备注 field.remark
6. 是否可编辑 field.readOnly
7. 是否在列表中显示 field.tableVisible

8. 关联字段显隐 field.relate
9. 关联字段选项值显隐 field.option.relate

数据结构

1. templateId => template-data
2. relateId => template-relate
3. dataId => data
